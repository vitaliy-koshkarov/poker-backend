## Backend service for Poker game

Poker game according to the rules of Texas Hold'em

### Launch instruction
1. Compile the application with the `mvn clean package` command
2. Run the application using the command `mvn spring-boot:run`

### Technologies stack:
* Java 21
* Spring Boot 3 (Security, Web, Data, JPA, WebSocket, Messaging)
* PostgreSQL 16
* Hibernate 6
* Flyway 10
* JSON Web Token
* Log4j2
* Lombok
